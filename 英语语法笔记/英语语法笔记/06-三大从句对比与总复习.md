# 06-三大从句对比与总复习

---

## 一、三大从句体系总览

```
从句
├── 定语从句（形容词性从句）→ 修饰名词
│     ├── 关系代词：that / which / who / whom / whose
│     └── 关系副词：when / where / why
│
├── 名词性从句（一个句子当名词用）
│     ├── 主语从句 → 在句首/It作形式主语
│     ├── 宾语从句 → 在动词/介词后
│     ├── 表语从句 → 在be动词后
│     └── 同位语从句 → 在抽象名词后解释内容
│
└── 状语从句（副词性从句）→ 修饰动词/句子
      ├── 时间：when / while / as / before / after
      ├── 条件：if / unless / as long as
      ├── 原因：because / since / as / for
      ├── 让步：although / though / even if
      └── 目的：so that / in order that
```

---

## 二、三大从句对比

### 1. 定语从句 vs 同位语从句（最易混淆）

```
The news that she said made me happy.
                  ↑
            她说的消息（定语从句：修饰 news）

The news that she got married made me happy.
                  ↑
          她结婚的消息（同位语从句：解释 news 的内容）
```

| 对比点 | 定语从句 | 同位语从句 |
|:-------|:---------|:-----------|
| 从句性质 | 形容词性（修饰） | 名词性（解释说明） |
| that 是否作成分 | ✅ 作主语/宾语 | ❌ 不作成分，只起连接 |
| that 能否省略 | 作宾语时可省略 | ❌ 不能省略 |
| 先行词类型 | 任何名词 | 抽象名词（news/fact/idea/truth…） |

### 2. 名词性从句内部的区别

```
主语从句：What he did is surprising.
宾语从句：I know what he did.
表语从句：The problem is what we should do.
同位语从句：The question what we should do bothers us.
```

|:-------|:--------:|:----------:|:--------:|
| **that** | ✅（人/物） | ✅（不作成分） | ❌ |
| **which** | ✅（物） | ✅ | ❌ |
| **who** | ✅（人） | ✅ | ❌ |
| **whom** | ✅（人，宾格） | ✅ | ❌ |
| **whose** | ✅（…的） | ✅ | ❌ |
| **when** | ✅（时间先行词） | ✅ | ✅ |
| **where** | ✅（地点先行词） | ✅ | ✅ |
| **why** | ✅（reason） | ✅ | ✅ |
| **if / whether** | ❌ | ✅（"是否"） | ✅（"如果"） |
| **unless** | ❌ | ❌ | ✅ |
| **because** | ❌ | ❌ | ✅ |
| **although** | ❌ | ❌ | ✅ |

## 四、谓语动词时态速查表

| 时态         | 构成                    | 关键词                   | 用法           |
| :--------- | :-------------------- | :-------------------- | :----------- |
| **一般现在**   | do/does               | always/every/usually  | 习惯、真理、主将从现   |
| **一般过去**   | did                   | yesterday/ago/last    | 过去动作         |
| **一般将来**   | will do / be going to | tomorrow/next/in      | 将来动作         |
| **过去将来**   | would do              | 过去语境中说将来              | 间接引语         |
| **现在进行**   | am/is/are doing       | now/at the moment     | 正在进行         |
| **过去进行**   | was/were doing        | when + 一般过去           | 过去某时进行       |
| **现在完成**   | have/has done         | already/yet/for/since | 对现在有影响/持续到现在 |
| **过去完成**   | had done              | 过去的过去                 | 两个过去动作，先发生的用 |
| **现在完成进行** | have been doing       | for/since/all day     | 持续到现在的动作     |
|            |                       |                       |              |

## 五、非谓语动词（to do / doing / done）速查

| 功能 | to do | doing | done |
|:-----|:------|:------|:------|
| 主语 | ✅ 具体 | ✅ 一般 | ❌ |
| 宾语 | ✅ want/hope/plan | ✅ enjoy/finish/avoid | ❌ |
| 定语 | ✅ 将来 | ✅ 主动进行 | ✅ 被动完成 |
| 状语 | ✅ 目的 | ✅ 主动伴随 | ✅ 被动伴随 |
| 宾补 | ✅ tell/want/ask | ✅ see/hear/watch | ✅ have/get |

## 六、被动语态速查

| 时态 | 被动形式 |
|:-----|:---------|
| 一般现在 | am/is/are + done |
| 一般过去 | was/were + done |
| 一般将来 | will be + done |
| 现在进行 | is being + done |
| 过去进行 | was being + done |
| 现在完成 | have/has been + done |
| 过去完成 | had been + done |

## 七、易错点清单

|  #  | 易错点                                          | 正确理解                    |
| :-: | :------------------------------------------- | :---------------------- |
|  1  | `that` 在定语从句中作宾语可省略，同位语从句中不能省略               | 看 that 是否作成分            |
|  2  | 定语从句用 `which` 还是 `that`？有介词/逗号用 which        | 介词+which / 逗号+which     |
|  3  | `if` 和 `whether`：whether 可放句首/接 or not/to do | if 不能放句首                |
|  4  | 宾语从句语序：陈述句语序                                 | 不是疑问句语序                 |
|  5  | 主将从现：if/when 从句用一般现在，主句用将来                   | 不是两个都用将来                |
|  6  | 进行时被动：`is being done`                        | 容易漏掉 being              |
|  7  | 非谓语动词逻辑主语一致                                  | doing/done 的逻辑主语 = 句子主语 |
|  8  | -ing 和 -ed 形容词的区别                            | -ing 令人…；-ed 感到…        |
