# 04-定语从句

---

## 一、什么是定语从句？

**定语从句 = 用一个句子来修饰名词**

```
I like the girl who comes from Hebei.
              └── 定语从句修饰 the girl
```

**结构：** 先行词 + 关系词 + 从句

```
The girl / who comes from Hebei / is my friend.
 先行词   关系词       从句
```

---

## 二、关系词的分类

### 关系代词

| 关系代词 | 指代对象 | 在从句中作 |
|:---------|:---------|:-----------|
| **that** | 人 / 物 | 主语 / 宾语 |
| **which** | 物 | 主语 / 宾语 |
| **who** | 人 | 主语 |
| **whom** | 人 | 宾语 |
| **whose** | 人 / 物 | 定语（…的） |

### 关系副词

| 关系副词 | 先行词类型 | 在从句中作 |
|:---------|:-----------|:-----------|
| **when** | 时间名词 | 时间状语 |
| **where** | 地点名词 | 地点状语 |
| **why** | reason | 原因状语（= for which） |

---

## 三、关系词的判断方法

```
① 看先行词是人还是物？
② 看关系词在从句中作什么成分？
   ┌─ 主语/宾语 → 关系代词（that/which/who/whom）
   │             时间 → when
   ├─ 状语 →     地点 → where
   │             原因 → why（reason）
   └─ 定语 → whose
```

---

## 四、⭐ that vs which 的区别

### 只能用 that 不能用 which 的情况（4种）

| 情况 | 示例 |
|:-----|:------|
| **① 先行词是不定代词** | `anything / something / nothing / everything / all / much / little / none / no` 后面只用 `that` |
| **② 先行词被最高级/序数词修饰** | `the best / the most / the first / the only / the last / the very` 后面只用 `that` |
| **③ 先行词既有人又有物** | `Let's talk about the person and the things that we remember.` |
| **④ 先行词是 that/those** | `What's that which is under the desk?` → 但 `that / those` 作先行词时，后跟 `which / who` |

```
This is the very book that I want to buy.       ← the very 只用 that
All that we need is time.                       ← all 只用 that
```

### 只能用 which 不能用 that 的情况

| 情况 | 示例 |
|:-----|:------|
| **介词 + which** | `This is the town in which I live.` |
| **非限制性定语从句** | 逗号后只用 `which / who`，不用 `that` |

---

## 五、⭐ 介词 + which / whom

```
This is the town in which I live.           （live in the town → in which）
This is the person of whom I am proud.       （be proud of → of whom）
```

**判断方法：** 把先行词放回从句中，需要什么介词就是什么介词

---

## 六、as 引导的定语从句

`as` 可以引导定语从句，常用于：
```
as we all know      （众所周知）
as is known to all  （众所周知）
as we expected      （正如我们所料）
```

---

## 七、限制性 vs 非限制性定语从句

| 对比 | 限制性定语从句 | 非限制性定语从句 |
|:-----|:--------------|:----------------|
| 符号 | 无逗号 | 有逗号隔开 |
| 作用 | 不能省略，去掉后句意不完整 | 可省略，补充说明 |
| 关系词 | that / which / who 等 | **不能用 that**，用 which / who |

```
The girl who is singing is my sister.        ← 限制性（哪个女孩？唱歌的那个）
My sister, who is singing, is very kind.    ← 非限制性（补充说明）
```
