# 03-非谓语动词（to do / doing / done）

---

## 一、概览

非谓语动词 = **不能作谓语的动词形式**，包括：

| 形式 | 含义 | 语态 |
|:-----|:-----|:------|
| **to do**（不定式） | 目的 / 将来 / 具体某一次 | 主动 |
| **to be done**（不定式被动） | 目的 / 将来 + 被动 | 被动 |
| **doing**（现在分词/动名词） | 主动 / 进行 / 习惯 | 主动 |
| **being done**（被动进行） | 正在被… | 被动 |
| **done**（过去分词） | 完成 / 被动 | 被动 |

---

## 二、非谓语动词的成分功能

### 1. 作主语

```diff
- 动词原形不能作主语！
+ Learning is important.        （doing 作主语）
+ To have higher scores means...（to do 作主语）
```

> **对比：** `Learning is important.` 强调一般性动作；`To learn well is important.` 强调特定目标。

### 2. 作定语

```diff
- 一个动词不能直接放名词前作定语
+ The boy learning English over here is my brother.       （doing 作定语，主动）
+ The books written for teaching chess came out.          （done 作定语，被动）
+ There is no reason to doubt his words.                  （to do 作定语）
+ a red apple                                              （形容词作定语）
+ An apple on the table                                    （介词短语作定语）
```

### 3. 作状语

⭐ **逻辑主语一致原则：** 非谓语动词的逻辑主语 = 句子主语

```diff
- ______from the mountain, we see our beautiful school.   → Seeing（we 主动看）
- ______from the mountain, our school looks beautiful.    → Seen（school 被看）
```

**判断方法：**
```
句子主语与非谓语动词之间是主动关系 → doing
句子主语与非谓语动词之间是被动关系 → done
```

**to do 作目的状语：**
```
To catch the bus, we get up early.
```

### 4. 作宾语

```
I like playing games.            ← 动名词作宾语
I want to go home.               ← 不定式作宾语
```

### 5. 作表语

```
Learning is important.           ← 动名词作表语
The best way is to do sth.       ← 不定式作表语
```

---

## 三、句型：It is + adj. + for / of sb. + to do sth.

```
It is important for us to learn.       ← for（事情的性质）
It is kind of you to help me.          ← of（人的品质）
```

**用 of 的形容词：** `kind / nice / polite / rude / clever / foolish / careful`

**用 for 的形容词：** `important / necessary / possible / hard / easy`

---

## 四、doing vs done 作形容词的区分

| 现在分词（doing / -ing） | 过去分词（done / -ed） |
|:------------------------|:----------------------|
| 令人…的（主动，修饰物） | 感到…的（被动，修饰人） |
| **interesting** film     | feel **interested**    |
| **exciting** news        | feel **excited**       |
| **boring** movie         | feel **bored**         |
| **tiring** work          | feel **tired**         |
| **frightening** scene    | feel **frightened**    |

```
I feel interested.          ← 人感到有趣
This film is interesting.    ← 电影本身有趣
```

## 五、非谓语动词速查表

| 功能 | to do | doing | done |
|:-----|:------|:------|:------|
| **主语** | ✅ To learn is important. | ✅ Learning is important. | ❌ |
| **宾语** | ✅ I want **to go**. | ✅ I enjoy **reading**. | ❌ |
| **表语** | ✅ My job is **to teach**. | ✅ My job is **teaching**. | ✅ The door is **locked**. |
| **定语** | ✅ a book **to read** | ✅ a boy **standing** there | ✅ a book **written** by him |
| **状语** | ✅ **To catch** the bus… | ✅ **Seeing** this, I cried… | ✅ **Seen** from hill… |
| **宾补** | ✅ tell him **to come** | ✅ see him **coming** | ✅ have it **fixed** |
