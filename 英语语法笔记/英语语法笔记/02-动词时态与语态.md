# 02-动词时态与语态

---

## 📌 总览：12 种时态

| 时态 | 构成 | 时间关键词 |
|:-----|:-----|:-----------|
| ⭐ **一般现在时** | do / does | always / usually / often / every + n. |
| ⭐ **一般过去时** | did | yesterday / ...ago / last... / in 1900 / just now |
| ⭐ **一般将来时** | will + do / be going to | tomorrow / next... / in + 时间段 |
| **过去将来时** | would + do | 在过去的语境中谈论未来 |
| ⭐ **现在进行时** | am / is / are + doing | right now / at the moment |
| ⭐ **过去进行时** | was / were + doing | 过去某时刻正在发生 |
| ⭐ **现在完成时** | have / has + done | already / yet / for / since / so far |
| ⭐ **过去完成时** | had + done | "过去的过去" |
| **现在完成进行时** | have / has + been doing | 从过去一直持续到现在 |
| **被动语态** | be + done（各种时态） | 强调动作承受者 |

---

## 三、一般现在时（do / does）

### 构成
```
I / You / 复数主语 → 动词原形
He / She / It → 动词 + s / es
```

### 用法1：经常性、习惯性动作
**标志词：** `always / sometimes / usually / often / every + n.`

```
We plant trees every March.
```

### 用法2：客观真理 / 事实

```
The sun rises in the east.
```

> **⚠️ 易错点：** 即使主句是过去时，客观真理依然用一般现在时！
> "My mom said the sun **rises** in the east."

### 用法3：在时间/条件状语从句中表将来
**主将从现原则：** 主句用将来时，从句（if / unless / when / as soon as / before / after / until / once / even if）用一般现在时。

```
If it **rains** tomorrow, we **will cancel** the class.
I will tell you the truth as soon as I **see** him.
```

---

## 四、一般过去时（did）

### 构成
```
动词 + ed（规则变化）/ 不规则变化
```

### 用法
表示过去某个时间发生的动作或存在的状态。

**标志词：** `yesterday / ...ago / last... / in + 过去年份 / in the past / the other day / just now`

```
When I was a freshman last year, I joined 4 clubs.
```

---

## 五、一般将来时（will + do）

### 构成1：will / shall + 动词原形
```
All of us will have a wonderful time in Shijiazhuang.
```

### 构成2：be going to + 动词原形
表示计划、打算或已有迹象要发生的事。

### 构成3：be doing（位移动词）
表示按计划或安排即将发生的动作。

**位移动词：** `go / come / leave / arrive / fly / start / move`

```
I am leaving Beijing tomorrow.（明天就要离开北京）
```

### 构成4：一般现在时表将来
用于时刻表、日历等。

```
Classes start at 6:00.
```

**时间标志词：** `tomorrow / in the future / next... / in a moment / in + 时间段（in 2 hours）`

---

## 六、过去将来时（would + do）

从过去的某个时间点看将要发生的动作。

```
She said that she **would give** me a computer in 2028.
```

---

## 七、现在进行时（am / is / are + doing）

### 用法
表示现在正在进行的动作。

**标志词：** `right now / at the moment`

```
I am reading a book.
```

---

## 八、过去进行时（was / were + doing）

### 用法
表示过去某个时刻正在进行的动作。

```
I was reading a book when he came in.
I was studying English when a friend knocked at the door.
```
（当...进来的时候，我正在读书）

---

## 九、现在完成时（have / has + done）⭐⭐⭐

### 用法1：过去发生，对现在有影响

```
I have lost my key.（现在还没找到）
```

### 用法2：过去开始，持续到现在

```
I have lived here for 3 years.
```

### 用法3：特殊句型

```
This is the first time that I have seen such a good movie.
```

**标志词：**
```
already / yet / lately / recently
in recent years
for + 时间段 / since + 时间点
so far / up to now / till now
in / during the past few years
```

---

## 十、过去完成时（had + done）⭐

表示"过去的过去"——在过去的某个时间点之前已经完成的动作。

```
When I arrived at the station, the bus **had left** already.
```
（我到达时，车已经开走了——"到达"是过去，"离开"发生在到达之前）

---

## 十一、现在完成进行时（have / has + been doing）

表示从过去开始，一直持续到现在，并且仍在进行。

```
We have been learning English.
```

---

## 十二、被动语态

### 构成

| 时态 | 被动形式 | 示例 |
|:-----|:---------|:------|
| 一般现在时 | am/is/are + done | The room is cleaned. |
| 一般过去时 | was/were + done | The room was cleaned. |
| **一般将来时** | **will be + done** | His wedding **will be held** in 2028. |
| 过去将来时 | would be + done | ... |
| **现在进行时** | **is being + done** | The room **is being painted**. |
| 过去进行时 | was being + done | ... |
| **现在完成时** | **have/has been + done** | The work has been done. |
| 过去完成时 | had been + done | ... |

### 用法

1. 强调动作承受者，不关心或不知道动作执行者
2. 用 `by` 引出动作执行者：`be done by ...`

```
We can't live in this room now. Because the room is being painted by the workers.
```

### 特殊结构

**① have / get sth done（让某事被做）**
```
I **have** my watch **repaired**.（我把表拿去修了）
```

**② need / want / require / deserve + to be done = + doing**
```
The room needs to be cleaned. = The room needs cleaning.
```

**③ be worth doing（值得被做）**
```
The book is worth reading.
```
