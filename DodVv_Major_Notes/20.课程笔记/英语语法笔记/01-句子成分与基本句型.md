# 01-句子成分与基本句型

---

## 一、句子成分总览

### 1. 主语
一个句子的主体，通常由名词、代词、不定式、动名词或从句充当。

### 2. 谓语动词
句子的核心，表示动作或状态。

```
be 动词 → be / be doing / be done
实义动词 → 各种时态形式
情态动词 + 动词原形
```

### 3. 宾语
动作的承受者，位于及物动词或介词后面。

```
I like playing games.  ← 动名词作宾语
```

### 4. 表语
位于系动词（be / seem / taste / look 等）之后，说明主语的特征、身份、状态。

```
The cakes taste delicious.   ← 形容词作表语
He is kind.                  ← 形容词作表语
```

### 5. 宾语补足语（宾补）
对宾语进行补充说明，与宾语构成逻辑上的主谓关系。

```
We call him Li Ming.                 ← 名词作宾补
We should keep the room clean.       ← 形容词作宾补
He made me happy.                    ← 形容词作宾补
```

### 6. 定语（adj.）
修饰名词，说明名词的特征。

```
It is a red apple.   ← 形容词作定语
```

### 7. 状语（adv.）
修饰动词、形容词、副词或整个句子，表示时间、地点、方式、原因等。

```
副词           → -ly 结尾的词
介词短语作状语 → in the room / with my roommate / at 7 o'clock

We are learning English in the room at 7 o'clock carefully.
```

### 8. 同位语
对前面的名词或代词进行解释说明。

```
The news that she will get married makes me happy.
My friend, Lily, is kind.
```

## 二、五大基本句型

### 句型1：主谓（S + V）

```
I arrived.
```

### 句型2：主谓宾（S + V + O）

```
I eat delicious cakes.
```

### 句型3：主谓双宾（S + V + IO + DO）
间接宾语（人）+ 直接宾语（物）

常见双宾动词：`give / send / pass / lend / borrow / buy / show / tell / teach`

```
He gives me the cake.      ← me（IO）the cake（DO）
Teach me English.
```

### 句型4：主谓宾宾补（S + V + O + OC）

```
He made me happy.
```

### 句型5：主系表（S + Lv + P）

```
He is kind.
```

## 三、连词

| 连词 | 含义 | 用法 |
|:-----|:-----|:------|
| **and** | 和、而且 | 连接并列成分 |
| **or** | 或者、否则 | 选择或否定条件 |
| **but** | 但是 | 转折 |
| **so** | 所以 | 因果 |
| **because** | 因为 | 原因 |
| **for** | 因为 | 补充原因（正式） |
| **therefore** | 因此 | 结果（正式） |
| **however** | 然而 | 转折（正式） |
| **while** | 当…时 / 然而 | 时间 / 对比 |
| **whereas** | 然而 | 对比（=while） |
| **as well as** | 以及 | 附加 |

## 四、思维导图

```
句子成分
├── 主语（名词/代词/不定式/动名词/从句）
├── 谓语动词（be/实义动词/情态动词）
├── 宾语（动作承受者）
├── 表语（系动词后）
├── 宾补（宾语后补充说明）
├── 定语（修饰名词）
├── 状语（修饰动词/句子）
└── 同位语（解释说明名词）

五大句型
├── 主谓
├── 主谓宾
├── 主谓双宾（人 + 物）
├── 主谓宾宾补
└── 主系表
```
