# 05-名词性从句（主语·宾语·表语·同位语）

---

## 一、什么是名词性从句？

**名词性从句 = 一个句子当名词用**

名词能作什么成分，名词性从句就能作什么成分：

```
What I am reading is interesting.    ← 主语从句（整个句子当主语）
I believe what he said.              ← 宾语从句（整个句子当宾语）
```

**四大名词性从句：**

| 从句类型 | 作用 | 相当于 |
|:---------|:-----|:-------|
| **主语从句** | 作主语 | 一个名词作主语 |
| **宾语从句** | 作宾语 | 一个名词作宾语 |
| **表语从句** | 作表语 | 一个名词作表语 |
| **同位语从句** | 解释说明名词 | 一个名词解释另一个名词 |

---

## 二、主语从句

### 1. 常见句型（⭐ 重点记）

| 句型 | 含义 |
|:-----|:------|
| **It is true that…** | …是真的 |
| **It is possible that…** | …是可能的 |
| **It is clear that…** | …很清楚 |
| **It is a fact that…** | …是事实 |
| **It seems / appears that…** | 似乎… |
| **It is said that…** | 据说… |
| **It is reported that…** | 据报道… |
| **It is suggested that…** | 有人建议… |
| **It is estimated that…** | 据估计… |
| **It is no wonder that…** | 难怪… |
| **It occurs to me that…** | 我突然想到… |

> **注意：** `It` 是形式主语，真正的主语是后面的 `that` 从句。

### 2. wh- 词引导的主语从句

```
What he said is important.
Whether he comes matters a lot.
```

---

## 三、宾语从句

### 1. 连接词

| 连接词 | 在从句中作 |
|:-------|:-----------|
| **that** | 不作成分（可省略） |
| **whether / if** | "是否"，不作成分 |
| **wh- 词**（what / which / who） | 主语/宾语 |
| **wh-ever 词**（whatever / whoever） | 主语/宾语 |
| **when / where / why / how** | 状语 |

```
I think that he is right.
I want to know what he is doing.
He asked whether I could help.
```

### 2. 宾语从句的语序

**从句用陈述句语序（主 + 谓），不用疑问句语序：**

```diff
+ I want to know what he is doing.      ← 陈述语序
- I want to know what is he doing.      ← ❌ 疑问语序
```

---

## 四、表语从句

放在系动词（be / seem / look 等）后面作表语。

```
The problem is that we don't have enough time.
That is why I was late.
The question is whether he will come.
```

---

## 五、同位语从句

### 1. 概念

**同位语 = 用一个句子解释一个抽象名词的内容**

常用于：`idea / fact / news / belief / thought / truth / proposal / problem / evidence`

```diff
The news / that she said / made me happy.         ← 定语从句（她说的那个消息）
The news / that she got married / made me happy.   ← 同位语从句（她结婚这个消息）
```

### 2. ⭐ 同位语从句 vs 定语从句

| 对比 | 同位语从句 | 定语从句 |
|:-----|:----------|:---------|
| **从句作用** | 解释名词的内容 | 修饰、限定名词 |
| **that 的作用** | 只起连接作用，不作成分（但不能省略） | 作主语/宾语（作宾语时可省略） |
| **被修饰的名词** | 抽象名词（news/fact/idea…） | 任何名词 |
| **去掉从句** | 句子依然完整（从句是额外内容） | 句子可能不完整 |

```
The news that he told me is true.               ← 定语从句（that 作 told 的宾语）
The news that he won the game is true.           ← 同位语从句（解释 news 的内容）
```

## 六、四类名词性从句速查

| 从句 | 位置 | 例句 |
|:-----|:-----|:------|
| **主语从句** | 句首 / It 形式主语 | **What he said** is true. / **It is true that** he said it. |
| **宾语从句** | 动词 / 介词后 | I know **what he did**. |
| **表语从句** | be 动词后 | The truth is **that he lied**. |
| **同位语从句** | 抽象名词后 | The fact **that he lied** shocked me. |
