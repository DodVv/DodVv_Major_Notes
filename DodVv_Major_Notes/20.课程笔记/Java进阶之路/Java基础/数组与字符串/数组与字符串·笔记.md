## 一、数组（Array）

### 1. 一维数组

**定义方式：**
```java
// 推荐写法
int[] arr1 = {1, 2, 3};            // 静态初始化
int[] arr2 = new int[]{1, 2, 3};   // 标准静态初始化
int[] arr3 = new int[5];           // 动态初始化（默认值0）
```

**数组长度和索引：**
```java
int[] arr = {10, 20, 30};
System.out.println(arr.length);   // 3
System.out.println(arr[0]);       // 10（第一个元素）
System.out.println(arr[arr.length - 1]);  // 30（最后一个元素）
```

**数组遍历：**
```java
// 标准for
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}

// 增强for
for (int n : arr) {
    System.out.println(n);
}
```

### 2. 二维数组

```java
// 定义方式
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

System.out.println(matrix[1][2]);  // 6
```

**打印二维数组：**
```java
// 使用 Arrays.deepToString()
System.out.println(Arrays.deepToString(matrix));
// [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```

### 3. 数组工具类 Arrays

| 方法 | 说明 |
|:-----|:-----|
| `Arrays.toString(arr)` | 将数组转成字符串 |
| `Arrays.deepToString(arr)` | 将多维数组转成字符串 |
| `Arrays.sort(arr)` | 升序排序 |
| `Arrays.binarySearch(arr, key)` | 二分查找（必须先排序） |
| `Arrays.copyOf(arr, newLength)` | 拷贝数组并指定新长度 |
| `Arrays.copyOfRange(arr, from, to)` | 拷贝指定范围 |
| `Arrays.equals(arr1, arr2)` | 比较两个数组内容是否相等 |
| `Arrays.fill(arr, value)` | 用指定值填充数组 |

## 二、字符串（String）

### 1. String 的特性

**不可变性**：String 类被 `final` 修饰，所有字符串内容存储在 `private final char value[]` 数组中，一旦创建就不能改变。

```java
String s = "二哥";
s = s + "的Java进阶之路";  
// 实际上创建了新对象，原对象"二哥"仍在常量池中
```

**两种创建方式的区别：**
```java
String s1 = "二哥";           // 存储在字符串常量池
String s2 = new String("二哥");  // 存储在堆内存
String s3 = "二哥";

System.out.println(s1 == s2);  // false（地址不同）
System.out.println(s1 == s3);  // true（常量池复用）
System.out.println(s1.equals(s2)); // true（内容相同）
```

> **⚠️ 二哥强调**：判断字符串内容相等一定要用 `equals()`，不要用 `==`！

### 2. String 常用方法

| 方法 | 说明 |
|:-----|:-----|
| `length()` | 获取长度 |
| `charAt(i)` | 获取第 i 个字符 |
| `toCharArray()` | 转成字符数组 |
| `equals(obj)` | 比较内容是否相等 |
| `equalsIgnoreCase(str)` | 忽略大小写比较 |
| `contains(str)` | 是否包含某子串 |
| `startsWith(prefix)` | 是否以某前缀开头 |
| `endsWith(suffix)` | 是否以某后缀结尾 |
| `indexOf(str)` | 返回子串首次出现的位置 |
| `lastIndexOf(str)` | 返回子串最后出现的位置 |
| `substring(begin, end)` | 截取 [begin, end) |
| `replace(old, new)` | 替换所有匹配字符/字符串 |
| `split(regex)` | 按正则分割成字符串数组 |
| `trim()` | 去除首尾空格 |
| `toUpperCase()` / `toLowerCase()` | 转大写/小写 |
| `isEmpty()` | 判断是否为空字符串 |

### 3. StringBuilder（高效字符串拼接）

```java
StringBuilder sb = new StringBuilder();
sb.append("Java").append(" 进阶之路").append("!");
String result = sb.toString();  // "Java 进阶之路!"
```

**String vs StringBuilder 性能对比：**
- `String` 的 `+` 拼接：每次都会创建新对象，**适合少量拼接**
- `StringBuilder`：在同一个对象上修改，**适合大量拼接**

> **二哥建议**：在循环中拼接字符串时，一定要用 StringBuilder！

### 4. StringBuffer

与 StringBuilder 类似，但 **StringBuffer 是线程安全**的（方法用 synchronized 修饰），StringBuilder 更快。

## 三、StringJoiner（JDK8+）

```java
StringJoiner sj = new StringJoiner(", ", "[", "]");
sj.add("Java");
sj.add("Python");
sj.add("Go");
System.out.println(sj);  // [Java, Python, Go]
```
