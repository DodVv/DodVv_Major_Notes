## 💡 学习建议

| 阶段 | 目标 | 方法 |
|:----|:-----|:-----|
| **第一阶段**（1-2周） | 掌握基础语法和面向对象 | 配合课堂代码，每个知识点自己写 demo |
| **第二阶段**（2-3周） | 掌握集合、IO、异常等核心 API | 理解底层原理，不要死记硬背 |
| **第三阶段**（持续） | 深入并发和 JVM | 结合面渣逆袭系列，理论与实践结合 |
