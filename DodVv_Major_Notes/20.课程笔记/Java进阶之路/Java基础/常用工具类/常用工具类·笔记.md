# 常用工具类

---

## 一、Arrays

```java
Arrays.toString(arr)           // 数组转字符串
Arrays.sort(arr)               // 排序
Arrays.binarySearch(arr, key)  // 二分查找
Arrays.copyOf(arr, len)        // 拷贝
Arrays.fill(arr, val)          // 填充
```

## 二、Objects（JDK7+）

```java
Objects.equals(a, b)           // 安全的 equals（避免空指针）
Objects.isNull(obj)            // 判断是否为 null
Objects.requireNonNull(obj)    // 非空检查
```

## 三、Collections

```java
Collections.sort(list)         // 排序
Collections.shuffle(list)      // 打乱
Collections.reverse(list)      // 反转
Collections.addAll(list, ...)  // 批量添加
```

## 四、Hutool（第三方工具库）

```java
StrUtil.isEmpty(str)           // 字符串判空
DateUtil.parse("2024-01-01")   // 日期解析
NumberUtil.add(a, b)           // 安全的加法
```

> **二哥建议**：Hutool 是一个国产的 Java 工具库，包含了大量常用工具方法，能显著提高开发效率。
