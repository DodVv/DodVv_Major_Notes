# 八股文 · 面试高频题

> 涵盖 Java 基础、集合、并发、JVM、Spring、MySQL、Redis 等高频面试题。

---

## 一、Java 基础

1. **面向对象三大特性？** 封装、继承、多态
2. ** == 和 equals() 区别？** 基本类型比值，引用类型比地址；equals 默认比地址，String 等已重写比内容
3. **String 为什么是不可变的？** final 修饰类 + private final char[] value
4. **StringBuilder 和 StringBuffer 区别？** 前者线程不安全（更快），后者线程安全
5. **重载和重写的区别？** 重载同类不同参，重写父子类同方法
6. **抽象类和接口的区别？** 抽象类 is-a（单继承），接口 like-a（多实现）
7. **final 关键字的作用？** 类不可继承、方法不可重写、变量只能赋值一次

## 二、集合

1. **HashMap 的底层原理？** 数组+链表+红黑树，put 流程：hash→桶→插入/覆盖→扩容
2. **ArrayList 和 LinkedList 区别？** 数组 vs 双向链表，查询/增删性能不同
3. **ConcurrentHashMap 安全机制？** JDK7 分段锁，JDK8 synchronized+CAS
4. **HashSet 如何去重？** 底层 HashMap，先 hashCode() 再 equals()

## 三、并发

1. **synchronized 原理？** JDK6 后锁升级（偏向→轻量→重量）
2. **volatile 作用？** 可见性 + 禁止重排序，不保证原子性
3. **线程池参数？** corePoolSize、maximumPoolSize、keepAliveTime、workQueue、handler

## 四、JVM

1. **内存区域划分？** 堆、方法区（元空间）、虚拟机栈、本地方法栈、PC
2. **GC 判断对象可回收？** 可达性分析（GC Roots）
3. **类加载机制？** 加载→验证→准备→解析→初始化，双亲委派

## 五、Spring

1. **IoC 是什么？** 控制反转，由容器管理对象生命周期和依赖
2. **AOP 的应用场景？** 日志、事务、权限、性能监控
3. **SpringBoot 自动配置原理？** @EnableAutoConfiguration + spring.factories

## 六、MySQL

1. **事务隔离级别？** RU→RC→RR（默认）→Serializable
2. **索引类型？** B+Tree、聚簇索引、辅助索引、联合索引、最左前缀
3. **MVCC 原理？** 隐藏字段 + undo log + ReadView

## 七、Redis

1. **五种数据类型？** String、List、Set、ZSet、Hash
2. **缓存穿透/击穿/雪崩？** 布隆过滤器、互斥锁、过期随机化
3. **持久化方式？** RDB（快照） + AOF（日志）
