# 💻 计算机基础 · 总览

---

## 一、操作系统

### 进程 vs 线程
| 对比 | 进程 | 线程 |
|:----|:-----|:-----|
| 资源 | 独立资源空间 | 共享进程资源 |
| 切换开销 | 大 | 小 |
| 通信 | IPC（管道/消息队列/共享内存） | 直接读写共享变量 |

### 死锁的四个必要条件
1. **互斥**：资源一次只能一个线程使用
2. **持有并等待**：持有资源的同时等待其他资源
3. **不可剥夺**：资源只能自愿释放
4. **循环等待**：形成循环等待链

## 二、计算机网络

### TCP 三次握手
```
Client → Server: SYN=1, Seq=x
Server → Client: SYN=1, ACK=1, Seq=y, Ack=x+1
Client → Server: ACK=1, Seq=x+1, Ack=y+1
```

### TCP 四次挥手
```
Client → Server: FIN=1
Server → Client: ACK=1
Server → Client: FIN=1
Client → Server: ACK=1
（Client 等待 2MSL 后关闭）
```

### HTTP 状态码
| 状态码 | 含义 |
|:------|:-----|
| 200 | OK 成功 |
| 301 | 永久重定向 |
| 302 | 临时重定向 |
| 400 | 客户端请求有误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
| 502 | 网关错误 |
| 503 | 服务不可用 |

### GET vs POST
| 对比 | GET | POST |
|:----|:-----|:------|
| 数据位置 | URL 参数 | 请求体 |
| 安全性 | 低（参数可见） | 较高 |
| 长度限制 | 有（URL 最大长度） | 无 |
| 幂等性 | ✅ 幂等 | ❌ 不幂等 |
| 缓存 | ✅ 可缓存 | ❌ 不缓存 |
