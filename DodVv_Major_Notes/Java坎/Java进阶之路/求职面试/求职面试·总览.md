# 🎯 求职面试 · 总览

> 面试不仅仅考八股文，更重要的是考察**解决问题的能力**和**技术深度**。

---

## 简历建议

1. **技术栈清晰**：列出熟练掌握的技术
2. **项目亮点突出**：用 STAR 法则描述（场景-任务-行动-结果）
3. **量化成果**：用数据说话（如"优化后接口响应时间降低 50%"）

---

## 面试准备路线

| 阶段 | 内容 | 时间 |
|:----|:-----|:----:|
| 基础突击 | Java基础、集合、JVM、并发 | 1-2周 |
| 框架复习 | Spring、SpringBoot、MyBatis | 1周 |
| 数据库 | MySQL、Redis 原理和优化 | 1周 |
| 项目深挖 | 准备2-3个核心项目 | 1周 |
| 算法刷题 | LeetCode 高频 100 题 | 持续 |
| 模拟面试 | 找人模拟，查漏补缺 | 3天 |
